Campo "About / Website" o pinned comment. Texto definitivo:

THEKEY
THE KING OF CHECKMATE

Governed Codex Transactions for Coding Agents

THEKEY es el producto y núcleo de transacciones gobernadas. THE KING orquesta
fases y contexto, pero no puede autoaprobarse ni eludir el PolicyEngine.
CHECKMATE analiza riesgos antes de la ejecución y emite un veredicto, pero no
realiza escrituras físicas.

Cada cambio propuesto por un agente se ejecuta en un workspace aislado — nunca
sobre el código original. Solo se promueve al repositorio lo que pasa todos los
gates deterministas y recibe autorización de política verificable. Cada ciclo
deja una cadena de evidencia auditable en SQLite. 102 tests. Demo reproducible
en Windows 11 sin Docker, sin WSL, sin GPU.
