# THEKEY — Descripción larga del repositorio (v0.2.0 Public Preview)

Campo "About / Website" o pinned comment. Texto definitivo:

---

THEKEY es un motor de transacciones Git gobernadas para agentes de
programación. Cada cambio propuesto por un agente se ejecuta en un workspace
aislado — nunca sobre el código original. Solo se promueve al repositorio lo
que pasa todos los gates deterministas y recibe autorización de política
verificable. Cada ciclo deja una cadena de evidencia auditable en SQLite.
102 tests. Demo reproducible en Windows 11 sin Docker, sin WSL, sin GPU.

---

Variante corta (bio / pinned):

THEKEY — governed Git transactions for coding agents. Isolated workspaces,
deterministic gates, auditable evidence. Workflow isolation, not OS-level
sandboxing. 102 tests, reproducible on Windows 11 without Docker. v0.2.0
Public Preview.

---

Restricciones cumplidas:
- "workflow isolation, not OS-level sandboxing" presente.
- "deterministic policy authorization" usado (no "auto-approval").
- No promete prevención total de riesgos de agentes autónomos.
- NPSC no mencionado.
- Versión "v0.2.0 Public Preview".
