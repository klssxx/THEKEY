THEKEY — THE KING OF CHECKMATE

Governed Codex Transactions for Coding Agents.

THEKEY is the governed-transaction core. THE KING coordinates phases and
context without self-approval or PolicyEngine bypass, while CHECKMATE performs
adversarial pre-execution review without physical writes. The problem it
solves: agent-driven changes are usually an opaque
blur of "someone edited something, tests ran, we shipped." THEKEY makes the

are separate roles; changes happen only in an isolated workspace; and every
artifact is hashed so a third party can verify the decision and detect
tampering.

sandbox, and it does not promise total security. 102 tests, demo runs on
Windows 11 with no Docker/WSL/GPU.
