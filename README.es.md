# THEKEY — THE KING OF CHECKMATE

**Transacciones Codex gobernadas para cambios más seguros de agentes de código.**

Esta es la portada breve en español. La ruta completa para evaluación está en [English README](README.md) y en [JUDGES.md](JUDGES.md).

THEKEY une un plan declarado, la revisión previa CHECKMATE, autorización humana explícita, política determinista, gates y evidencia persistida. El aislamiento de workflow protege el árbol fuente, pero no es un sandbox de sistema operativo.

y abre `THEKEY.exe` en Windows 10/11 x64. Consulta las instrucciones completas en [JUDGES.md](JUDGES.md) y [la guía rápida en español](docs/build-week/JUDGE_QUICKSTART.es.md).
